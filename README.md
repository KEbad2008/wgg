# Welcome to the Word guessing Game by @ebaduk117

this game is written in python, using my new skills i learned on freecodecamp


basically, users are prompted with a menu and can see instructions, exit, or play the game.

when playing, players guess the letters and try to guess the generated word.

players have 5 incorrect guesses before their session is terminated.

for more details use the instructions option on the game menu.

demo link: https://codesandbox.io/p/devbox/wjmmgx. go there, click run button to use program.
